* virtual enviornment
create:   	python -m venv venv_name
activate  	venv_name\Scripts\activate
deactivate:	deactivate 
remove:  	rmdir /s /q venv_name 

* save/restore configuration with;
pip freeze > python_config.txt
pip install -r python_config.txt

* Example batch file to launch terminal with virtual enviornment
@echo off
REM Open CMD with virtual environment ready
start cmd.exe /K "C:\...\venv\Scripts\activate.bat"
